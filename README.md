Before you go to below step. Make sure that you downloaded and installed revel & revel cmd. If you didn't, check this out: `http://revel.github.io/tutorial/gettingstarted.html`.

1. open cmd
2. create local gdg project `mkdir gdg`
3. cd gdg
4. create src directory `mkdir src`
5. copy app directory above into src
6. set gopath
7. `run revel gdg`
